Bem-vindo ao Speech-to-Text App!

Transforme sua fala em texto com facilidade!

Sobre o Projeto

Este aplicativo utiliza a API Gemini para transcrever áudio em texto. Com funcionalidades para gravar áudio, converter formatos e baixar áudio do YouTube, ele é uma ferramenta poderosa para quem precisa de transcrições rápidas e precisas.

Funcionalidades

  • Gravação de áudio do microfone
  • Transcrição de arquivos de áudio existentes
  • Conversão de áudio para MP3
  • Integração com YouTube para download de áudio

Como Usar

  1. Escolha uma opção: gravar áudio, fornecer um arquivo ou um link do YouTube.
  2. Para gravação, clique no botão e fale.
  3. Para arquivos, forneça o caminho completo.
  4. Para YouTube, cole o link e aguarde o download.
